- 什么是ROS
  collapsed:: true
	- 机器人界的安卓
	- 模块化，分布式
	- node类比安卓的app：建立数据连接（usb串口，单片机），然后node和物理设备通讯
- 为什么用ROS
  collapsed:: true
	- 可以应用不同的包实现不同的功能
	- 也可以发布新的包
- ubuntu简单使用
  collapsed:: true
	- 文本编辑器：gedit demo.txt
	- 执行文件中的指令序列：source demo.sh，连续执行一串指令
	- ~/.bashrc：终端程序启动脚本，使用ls -a才能看到，可以更改
	- sudo：以管理员权限执行本条指令
