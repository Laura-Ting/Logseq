			- 联动：一般用于二空题，三空题，根据逻辑（有时取同，有时取反，有时特定）

	- #### 指代重复
	  collapsed:: true
		- 定义
		  collapsed:: true
			- 对象的重复：指代词或者指代结构必然重复前文的同一个或同一类对象
			- 特征的重复：相同对象的特征取同
		- 怎样处理
		  collapsed:: true
			- 构成重复关系的对象或特征，彼此取同
		- 关键词
		  collapsed:: true
			- his, her, this, that, these, those, its, the former the latter the work's, such,the same,同位语，插入语等
		- 注意
		  collapsed:: true
			- 指代重复的特征有时候词性或表达形式不同，同时对象和特征之间也可能有重复关系
			  collapsed:: true
				- e.g.这种粒子寿命短暂，往往在一纳秒内就__了.adj->v
				- e.g.他是个懦夫，面对强敌总是表现出__的一面。n->adj
			- 指代重复不受取反逻辑的约束，取反逻辑不针对对象本身的属性，而是针对该对象的评价或态度！
			  collapsed:: true
				- e.g.小明很胖,但是他的__并不阻碍他跑得快。
